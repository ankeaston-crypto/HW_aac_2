8. flex-grow: разные значения

flex-grow: способность элемента увеличиваться при свободном месте.

flex-grow: 1 (растянется)
flex-grow: 2 (растянется в 2 раза больше)
flex-grow: 0 (не растянется)

Первый элемент растягивается, второй в 2 раза больше, третий не растягивается

9. flex-shrink: 0 (запрет сжатия)

flex-shrink: способность элемента уменьшаться при нехватке места.

flex-shrink: 0 (не сжимается)
flex-shrink: 1 (сжимается)
flex-shrink: 1 (сжимается)

Уменьши ширину окна - первый элемент не сжимается, остальные сжимаются

12. Order: изменение порядка элементов

меняет порядок отобреажения элементов. по умолчанию все элементы имеют order: 0. элементы с меньшим значением order отображаются раньше. Полезно для перестановки блоков на мобильных устройствах.

order: 3 (покажется последним)
order: 1 (покажется вторым)
order: 2 (покажется третьим)
order: 0 (покажется первым)

Порядок отображения: order:0 → order:1 → order:2 → order:3

Полезно, например, чтобы на мобильных экранах меню показывалось после основного контента
